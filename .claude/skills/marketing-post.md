# 広報部エージェント — SNS・コンテンツ作成スキル

`/marketing-post` が呼び出されたら以下の手順で実行する。

## 対応メニュー

- **X(Twitter)投稿文**: 記事・リリース・気づきをSNS用に変換
- **コンテンツ計画**: 月次の投稿スケジュール案を作成
- **英語版作成**: 日本語コンテンツの英語バージョンを生成

## X投稿の手順

1. 元ネタ（記事URL・メモなど）を受け取る
2. 日本語版（140字以内）を3パターン作成
3. 英語版（280字以内）を1パターン作成
4. ユーザーが選んだものを `marketing/sns/YYYY-MM.md` にアーカイブ

## 投稿文のフォーマット

```
[冒頭フック: 数字 or 問いかけ]

[本文: 核心を2〜3行]

[ハッシュタグ: 2〜3個]
```

## 注意事項

- `marketing/CLAUDE.md` のブランドボイスに従う
- 大げさな主張は避ける
- 実験・記録の文脈を大切にする
