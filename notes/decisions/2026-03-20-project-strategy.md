作成日: 2026-03-20
役割: このプロジェクトの目的・構造・技術選択の土台。フェーズが進んでも変わらない部分を書く。

## 何をするプロジェクトか

**一言で:** AIエージェントが各部門を担う「一人会社」を構築しながら、Claude Code の全機能を実践で習得する。

**2つの目的（同時並行）:**
1. 個人ウェブサイト `start-solopreneur.vercel.app` を育てる
2. Claude Code の全機能を作りながら習得する

## ディレクトリ＝会社組織

```
start_solopreneur/
│
├── CLAUDE.md          ← 社長指示書（全AIエージェント共通ルール）
├── src/               ← 【開発部】 Webアプリ本体
├── content/           ← 【編集部】 ブログ記事（MDX）
├── marketing/         ← 【広報部】 戦略・SNSテンプレ
├── finance/           ← 【財務部】 収益モデル・試算
├── notes/             ← 【社長メモ】 意思決定の記録
└── .claude/
    └── skills/        ← 各部門エージェントの役割定義
```

**設計の核心:**
- `CLAUDE.md`（ルート）= 社長から全社員への方針
- `CLAUDE.md`（各部門）= 部門長への個別指示
- `skills/` = 編集者・CFO・マーケターなど各担当AIの定義

## 技術スタック

| 用途 | 技術 | 選定理由 |
|------|------|---------|
| フレームワーク | Next.js 15 (App Router) | 情報量・将来性 |
| スタイル | Tailwind CSS | 開発速度 |
| 記事管理 | MDX + next-mdx-remote | コードと分離、柔軟性 |
| デプロイ | Vercel（無料枠） | コスト¥0 |
| リポジトリ | GitHub: sbilxxxx/start_solopreneur | Vercelとの連携 |
| 自動デプロイ | Stopフック → git push → Vercel自動検知 | 手動作業ゼロ |

---

## 決定済みの方針

- Webフレームワークは **Next.js**（情報量・将来性）
- デプロイは **Vercel無料枠**（ランニングコスト¥0）
- コードを変更したら **自動でGitHub push → 本番反映**
- 記事は **MDXファイル**で管理（コードと分離）
- メモは **notesディレクトリ** に日付付きで保存
- 収益化はPhase 6（見えてから考える。今は基盤構築に集中）
