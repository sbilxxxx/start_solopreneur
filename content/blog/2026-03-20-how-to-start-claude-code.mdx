「Claude Codeって、どこからどう始めるんだろう」

使い始める前、一番わからなかったのはここだった。「AIでコードを書ける」というのはわかった。でも、それをどうやって自分のPCで動かすのか、どこに何を書けばいいのか、そこが全然見えなかった。

この記事は、その疑問に答えるために書く。インストールから「サイトが画面に表示された」瞬間まで、自分がやったことをそのまま書く。

---

## 始める前に必要なもの

Claude Codeを使うには、2つが必要だ。

**1. Claude のサブスクリプション（または API キー）**

Claude Code は Anthropic（アンソロピック）という会社が作っている。使うには、[claude.ai](https://claude.ai) の有料プラン（Pro/Max）か、Anthropic の API キーが必要になる。

自分は Claude Pro のサブスクリプションを使っている。

**2. Node.js（バージョン 18 以上）**

Node.js（ノード・ジェイエス）は JavaScript を PC 上で動かすためのソフトウェアだ。Claude Code 自体が Node.js で動いているので、先にインストールしておく必要がある。

インストール済みかどうかはターミナルで確認できる：

```bash
node --version
```

`v18.0.0` 以上と表示されれば OK。表示されなければ [nodejs.org](https://nodejs.org) からダウンロードする。

---

## Claude Code のインストール

ターミナル（Mac なら「ターミナル」、Windows なら「PowerShell」や「コマンドプロンプト」）を開いて、以下を実行する：

```bash
npm install -g @anthropic-ai/claude-code
```

`npm install -g`（グローバルインストール）は「このコマンドを PC 全体で使えるようにする」という意味だ。

インストールが終わったら、動いているか確認する：

```bash
claude --version
```

バージョン番号が表示されれば成功。

---

## 最初の起動とログイン

プロジェクトを置きたい場所に移動して、`claude` と入力するだけで始まる。

```bash
mkdir my-first-site
cd my-first-site
claude
```

`mkdir`（メイクディレクトリ）はフォルダを作るコマンド。`cd` はそのフォルダに移動するコマンド。

初回起動時はブラウザが開いてログイン画面になる。Claude のアカウントでログインすれば、認証が完了してターミナルに戻ってくる。

---

## 最初の会話

ログインすると、こんな画面になる：

```
╭─────────────────────────────────────╮
│ ✻ Welcome to Claude Code!           │
╰─────────────────────────────────────╯

>
```

ここに日本語で話しかければいい。

最初に自分が入力したのはこれだった：

```
Next.jsでウェブサイトを作りたい。
まだ何も用意していない。セットアップから全部やって。
```

すると Claude Code が動き始めた。

```
Next.jsプロジェクトを作成します。

まず `npx create-next-app@latest` を実行して...
```

自分は何もしなくていい。Claude Code が npx（パッケージを一時的に実行するコマンド）を使ってプロジェクトのファイルを作り始める。少し待つと、フォルダの中にたくさんのファイルができている。

---

## 「動いているか確認して」と言うだけでいい

ファイルができたら、こう聞けばいい：

```
ローカルで動かして確認したい
```

Claude Code が `npm run dev` というコマンドを実行してくれる。`npm run dev` は「開発用にサイトを起動する」コマンドで、自分の PC 上だけで動くサイトが立ち上がる。

```
ローカルサーバーが起動しました。
http://localhost:3000 で確認できます。
```

ブラウザで `http://localhost:3000` を開く。Next.js のデフォルト画面が表示される。

これが最初の「動いた瞬間」だ。

自分が書いたコードは 0 行。ターミナルで 3 回コマンドを入力して、あとは Claude Code に話しかけただけで、画面にサイトが表示されている。

---

## その後のカスタマイズ

デフォルトの Next.js 画面が出たあとは、会話でカスタマイズしていく。

```
ホームページのタイトルを「Start Solopreneur」に変えて。
背景は白、テキストは黒でシンプルにして。
```

こう伝えると、Claude Code が `src/app/page.tsx` を開いて書き換えてくれる。ブラウザをリロードすると変更が反映されている。

この「指示 → 自動で変更 → ブラウザで確認」のサイクルが開発の基本だ。

---

## 本番公開（Vercel へのデプロイ）

ローカルで動いているサイトを外部からアクセスできる状態にするには、デプロイ（公開）が必要になる。

このサイトは Vercel（バーセル）という無料のホスティングサービスを使っている。デプロイの仕組みや設定方法は[別の記事](/blog/2026-03-20-auto-deploy-hook)に書いてあるので、そちらを参考にしてほしい。

---

始めてみると、思っていたより「始めやすい」と感じた。

難しかったのはインストールよりも、「最初に何を指示すればいいか」だった。Claude Code は高機能だけど、「何をお願いできるか」を知らないと最初は迷う。

この記事で書いた手順は、そのまま自分が最初にやったことだ。「まず動く状態にする」だけなら、これだけで十分だと思う。
